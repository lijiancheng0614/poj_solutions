# An Easy Problem!

## 题目

给定 n 进制的整数 r，r 能被 n - 1 整除

求最小的 n


## 方法

数学

枚举答案，判断每位数之和是否能被 n - 1 整除即可

简单的证明：

设 r = abcd，则

(a * n^3 + b * n^2 + c * n + d) % (n - 1) = 0

=> (a * n ^ 3 % (n - 1) + b * n^2 % (n - 1) + c * n % (n - 1) + d % (n - 1)) % (n - 1) = 0

=> ((a * n % (n - 1) * n % (n - 1) * n % (n - 1) + b * n % (n - 1) * n % (n - 1) + c * n % (n - 1) + d) % (n - 1) = 0

又 n % (n - 1) = 1

故 (a * 1 * 1 * 1 + b * 1 * 1 + c * 1 + d) % (n - 1) = 0

=> (a + b + c + d) % (n - 1) = 0
